Criao do DataFrame para armazenar a varivel

Os DataFrames so objetos bidimensionais, de tamanho varivel.

O seu formato de uma tabela, onde os dados so organizados em linhas e colunas.

Alm disso, enquanto podemos pensar a Series como uma nica coluna, o DataFrame seria uma unio de vrias Series sob um mesmo index.
